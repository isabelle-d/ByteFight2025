# cs3600Tournament
Each agent controls a worker on an 8x8 chessboard. The agents take turns making moves. After
both have had 40 turns, the game is terminated.
Each agent gets a total of 4 minutes to come up with its moves. (Not 4 minutes for each move. A
total of 4 minutes for all your moves. The game will not last longer than 8 minutes.) If a player
runs out of time, they lose the game.
The agent with the most points when the game terminates is the winner. You win points through
these activities
